# parser/

## Files

- `state_vars.py` — collect state variable names
- `function_extractor.py` — list implemented functions
- `modifier_extractor.py` — list modifiers
- `name_resolver.py` — resolve AST expression names/strings
- `function_summary.py` — aggregate detector flags per function
- `modifier_summary.py` — infer security properties from modifiers

## Responsibility

Provide normalized inputs for detectors and printers.
